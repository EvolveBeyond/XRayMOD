## XrayMOD چیست؟

XrayMOD یک **پنل مدیریت پروکسی سرورلس و خودمیزبان** است که به‌طور کامل روی Cloudflare Workers اجرا می‌شود. این پنل داشبورد کاملی برای مدیریت پروتکل‌های VLESS، Trojan و Shadowsocks با قابلیت‌های پیشرفته ضد سانسور، بهینه‌سازی IP تمیز و رابط کاربری مدرن تاریک فراهم می‌کند.

**نیازی به زیرساخت نیست** — بدون VPS، بدون دامنه، بدون Docker. فقط روی Cloudflare استقرار بدید و شروع کنید.

### تفاوت‌های کلیدی

- اجرای رایگان روی **پلن رایگان** Cloudflare بدون وابستگی خارجی
- سیستم **ضد شناسایی** با صفحات خطای جعلی Cloudflare
- **اسکنر IP تمیز** با بهینه‌سازی به‌تفکیک ISP برای شبکه‌های ایران
- **gRPC + XHTTP** علاوه بر WebSocket استاندارد
- **ECH + TLS Fragment** برای عبور پیشرفته از DPI
- **ربات تلگرام** برای مدیریت کامل پنل از موبایل
- **حالت بکند** — کاربران VPS خودشان رو ارائه میدن
- **تلگرام مینی اپ** — پنل واکنش‌گرا داخل تلگرام
- استقرار با یک کلیک از طریق اسکریپت نصب یا ویزار

## شروع سریع

### نصب با یک دستور (توصیه شده)

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/EvolveBeyond/XRayMOD/refs/heads/main/install.sh)
```

نصب‌کننده همه چیز رو انجام میده:
1. بررسی پیش‌نیازها (curl, uv)
2. دانلود اسکریپت استقرار
3. درخواست توکن API کلاودفلر
4. ایجاد خودکار پایگاه داده D1
5. استقرار ورکر
6. نمایش URL پنل و اطلاعات ورود

### استقرار با ویزار (برای فروشندگان)

ویزار رو یکبار استقرار بدید، سپس برای دیگران پنل بسازید:

```bash
git clone https://github.com/EvolveBeyond/XRayMOD.git
cd XRayMOD/wizard
wrangler deploy
```

### استقرار دستی

```bash
git clone https://github.com/EvolveBeyond/XRayMOD.git
cd XRayMOD
npm install
```

شناسه D1 خودتون رو در `wrangler.toml` وارد کنید و `npm run deploy` بزنید.

## سیستم پنهان‌سازی

پنل رو پشت مسیرهای مخفی پنهان میکنه و به بازدیدکنندگان غیرمجاز صفحات خطای جعلی Cloudflare نمایش میده.

### نحوه کار

| مرحله | اتفاق |
|-------|-------|
| ۱. مسیرهای مخفی | مسیرهای سفارشی برای ادمین، ورود، اشتراک |
| ۲. بازنویسی مسیر | بازدید از `/x7k9m` به `/admin` بازنویسی میشه |
| ۳. صفحات جعلی | بازدید مستقیم از `/admin` خطای ۱۱۰۱ جعلی نشون میده |
| ۴. جایگزین | هر آدرس ناشناخته‌ای صفحه جعلی نشون میده |

### متغیرهای محیطی

| متغیر | توضیحات | پیش‌فرض |
|--------|---------|---------|
| `DISGUISE_PAGE` | `1101` یا `nginx` | `1101` |
| `PANEL_RECOVERY` | `1` برای غیرفعال کردن پنهان‌سازی | `false` |

## اسکنر IP تمیز

تولید و مدیریت خودکار IP‌های کلاودفلر بهینه‌شده برای ISP شما.

### اپراتورهای پشتیبانی شده

| اپراتور | ASN | کد |
|---------|-----|-----|
| ایرانسل / MTN | 44244 | `mtn` |
| همراه اول (MCI) | 197207 | `mci` |
| رایتل | 57218 | `rightel` |
| شاتل | 31549 | `shatel` |

### نحوه استفاده

1. تب **IP تمیز** در پنل ادمین
2. کلیک **اسکن IP** — تولید IP تصادفی
3. بررسی نتایج و کلیک **اعمال بهترین IP‌ها**
4. لینک‌های اشتراک از IP‌های بهینه استفاده میکنن

## ربات تلگرام

مدیریت کامل پنل از تلگرام با ناوبری دکمه‌ای.

### راه‌اندازی

1. ساخت ربات از طریق [@BotFather](https://t.me/BotFather)
2. کپی توکن ربات
3. رفتن به تنظیمات → ربات تلگرام در پنل
4. وارد کردن توکن و Chat ID
5. ذخیره و اجرای دستور webhook

### دستورات

| دستور | توضیحات |
|-------|---------|
| `/start` | منوی اصلی با دکمه‌ها |
| `/status` | آپتایم، تعداد کاربران، اطلاعات پروتکل |
| `/config` | تنظیمات فعلی پروتکل و امنیت |
| `/sub` | دریافت لینک اشتراک |
| `/users` | لیست کاربران (ادمین) |

## حالت بکند

به کاربران اجازه بدید VPS خودشون رو برای کانفیگ شخصی ارائه بدن.

### نحوه کار

1. کاربر در بازار → سرور شما روی **ثبت** کلیک میکنه
2. آدرس IP VPS خودش رو وارد میکنه
3. اسکریپت نصب رو روی VPS اجرا میکنه
4. اسکریپت Xray-core رو با UUID کاربر نصب میکنه
5. لینک‌های اشتراک خودکار از طریق VPS مسیریابی میشن

## لینک‌های اشتراک

### فرمت‌ها

| فرمت | پارامتر | توضیحات |

## عیب‌یابی

### ورکر استقرار نمی‌یابد
- وارد بودن Wrangler: `wrangler whoami`
- شناسه D1 در `wrangler.toml`
- نصب وابستگی‌ها: `npm install`

### API خطای 401 برمی‌گرداند
- تنظیم `ADMIN_PASSWORD` در متغیرهای محیطی
- پاک کردن کوکی‌های مرورگر

### لینک اشتراک کار نمی‌کند
- فعال بودن وضعیت کاربر
- صحت UUID در URL
- نگذشتن تاریخ انقضا

### قفل شدن توسط پنهان‌سازی
- `PANEL_RECOVERY=1` در متغیرهای محیطی ورکر
